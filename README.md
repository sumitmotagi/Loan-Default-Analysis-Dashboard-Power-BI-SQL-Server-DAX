# Loan-Default-Analysis-Dashboard-Power-BI-SQL-Server-DAX
Built an end-to-end Loan Default Analysis Dashboard using Power BI, SQL Server, Power Query, and DAX. Developed Dataflows, automated refresh processes, and advanced DAX measures (YOY, YTD, Default Rate, Median). Analyzed borrower risk, loan performance, credit profiles, and default trends through interactive dashboards.
